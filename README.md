# interets_simples_compte_epargne
 Calculateur d'Intérêts Simples

Un programme Python qui calcule les intérêts simples pour les comptes d'épargne.

 Qu'est-ce que ce programme fait ?

Ce programme vous aide à calculer :
- Combien d'intérêts vous allez gagner sur votre épargne
- Combien de temps il faut pour atteindre un objectif d'épargne
- Quel taux d'intérêt vous avez besoin
- Combien d'argent déposer initialement

 Comment l'exécuter

1. **Téléchargez le fichier** `interets_simples_compte_epargne.py`

2. **Ouvrez votre terminal/invite de commandes**

3. **Lancez le programme :**
   ```bash
   python interets_simples_compte_epargne.py
   ```
   
   Ou sur certains ordinateurs :
   ```bash
   python3 interets_simples_compte_epargne.py
   ```

4. **Suivez le menu !** Le programme vous montrera les options 1-7. Tapez simplement un numéro et appuyez sur Entrée.

 Exemple Rapide

```
Choisissez l'option 1 pour calculer les intérêts :
- Entrez : 5000 (votre épargne)
- Entrez : 2.5 (taux d'intérêt %)
- Entrez : 3 (années)

Résultat : Vous gagnerez 375€ d'intérêts !
```

 Options du Menu

Quand vous lancez le programme, vous verrez :

1. **Calculer les intérêts** - Voir combien vous allez gagner
2. **Calculer la durée** - Combien de temps pour atteindre votre objectif
3. **Calculer le taux** - Quel taux vous avez besoin
4. **Calculer le montant initial** - Combien déposer
5. **Voir le tableau annuel** - Suivre la croissance année par année
6. **Comparer les taux** - Comparer différentes banques
7. **Voir les exemples** - Exemples prêts pour mieux comprendre
0. **Quitter** - Fermer le programme

 Prérequis

- Python 3.6 ou plus récent (c'est tout !)
- Aucune bibliothèque supplémentaire nécessaire

  
 Besoin d'Aide ?

Si le programme ne fonctionne pas :
1. Vérifiez que Python est installé : `python --version`
2. Vérifiez que vous êtes dans le bon dossier
3. Essayez `python3` au lieu de `python`
